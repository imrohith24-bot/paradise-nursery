# Paradise Nursery

## Project Description
Paradise Nursery is a dynamic, single-page e-commerce web application built with React and Redux. It allows users to browse a curated selection of houseplants, add them to a virtual shopping cart, and manage their selected items before checkout.

## Features
* **Landing Page:** Welcoming interface with a background image and a "Get Started" call to action.
* **Product Catalog:** Displays 18 unique houseplants categorized by their properties (Air Purifying, Aromatic, Insect Repellent).
* **Shopping Cart:** Users can add items, view their cart, adjust quantities, and remove items.
* **State Management:** Utilizes Redux Toolkit to seamlessly manage global cart state across all components.

## Technologies Used
* React.js
* Redux Toolkit (`react-redux`, `@reduxjs/toolkit`)
* CSS for styling
